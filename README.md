# Web Template #

This is my personal template to start a new front-end project.

```

## Folder Structure ##

```
.
├── css (CSS files)
│   ├── vendor (Vendor CSS files)
│   ├── fonts.css
│   └── style.css
├── fonts (Font files)
├── img (Images for layout)
│   └── icons (Icon files)
├── js (JavaScript files)
│   ├── vendor (Vendor JS, e.g. jQuery, Modernizr, ...)
│   │   ├── jquery.min.js
│   │   └── plugins.min.js
│   └── main.js
├── .htaccess
├── favicon.ico
└── index.php
```
